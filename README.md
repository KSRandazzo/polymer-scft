# polymer-scft

A minimal Python implementation of self-consistent field theory (SCFT) for polymer systems.

This project is a work-in-progress research codebase for SCFT simulations of polymers using simple numerical methods.

## Current status

- Early-stage development
- Basic SCFT solver (not yet implemented)
- Focus on 1D lattice systems

## Goals

- Build a clean SCFT solver in Python
- Separate physics (fields, propagators) from numerics
- Create reproducible polymer simulation examples
